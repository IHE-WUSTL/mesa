Test Description

The purpose of this test is to ensure that when an EC modifies an existing BSPS, it creates a new BSPS instance.

Test Setup

In this test, the Evidence Creator will modify the Blended Softcopy Presentation State (BSPS) object created in test 3540. Test 3540 must be run before this one. The Evidence Creator will modify the transparency of the SUPERIMPOSED dataset and create a new BSPS instance.
