Test Description

In this test, the Evidence Creator creates a valid BSPS DICOM object which references a Spatial Registration object.

Test Setup

The Evidence Creator accesses the 2 image sets and creates Spatial Registration object. MESA prompts the Evidence Creator to create a BSPS object which references the Spatial Registration object. MESA prompts the Evidence Creator to C-STORE the BSPS and Spatial Registration object to the MESA Image Manager.
