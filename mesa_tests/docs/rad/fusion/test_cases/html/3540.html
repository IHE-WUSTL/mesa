Test Description

In this test, the Evidence Creator creates a Blended Softcopy Presentation State (BSPS) object that specifies how to blend for display two series of images. These series were created on a hybrid Acquisition Modaity and have the same Frame of Reference. This tests transaction [RAD-57] in the Image Fusion Profile and Storage Commitment [RAD-10]. Spatial Registration is tested in another test

Test Setup

The Evidence Creator imports 2 series (1 PET, 1 CT) provided with the MESA distribution, prior to using its application to specify blending parameters (window width and center, color map and transparency) and encode them in a BSPS object. The Evidence Creator C-STOREs the BSPS object to the Image Manager and requests Storage Commitment.
