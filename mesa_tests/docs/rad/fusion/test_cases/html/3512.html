Test Description

In this test, the Evidence Creator use their application to determine the spatial transformation necessary between the two image sets having different Frame of Reference UIDs and create a Spatial Registration (SRI) object.

Test Setup

The Evidence Creator imports 2 series (1 PET, 1 CT) provided with the MESA distribution. These sets have different Frame of Reference UIDs. Based on the spatial transformation Evidence Creator creates a Spatial Registration (SRI) object. MESA prompts the Evidence Creator to C-STORE the Spatial Registration object to the MESA Image Manager.
